# Invitrack - a production-grade inventory management system with features like transaction handling, conflict resolution, distributed locking, and audit logging.

Features (Planned)

- Role-Based Access Control (RBAC)
- JWT Authentication + OAuth support
- Secure password hashing
- Resource booking system
- Priority scheduling (Manager > Employee)
- Concurrent request handling
- Database transaction management
- Audit logging
- Analytics dashboard
- Cloud deployment (AWS)

Tech Stack (Planned)

Backend:
- Node.js
- Express
- PostgreSQL
- Redis (for locking & caching)

Frontend:
- React

Infrastructure:
- AWS (EC2, S3)
- GitHub Actions (CI/CD)
